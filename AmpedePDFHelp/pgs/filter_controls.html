Filter Controls

PDF has five keyframeable controls in the Filters tab of the Viewer window. (A sixth control is used to flush Final Cut Pro's image cache.) These controls mirror similar controls in Final Cut Pro's Motion tab, but support continuous rasterization. You should use PDF's controls, not the controls in the Motion tab.


Vector Scale

This control allows you to scale the image. 100 means 100% scale; the image will fill the frame perfectly, possibly with space on the sides or the top if you have Preserve aspect ratio enabled.


Scale Modifier

This control allows you to increase the range of the Vector Scale control. PDF multiplies the Vector Scale amount by the Scale Modifier amount. By default, Scale Modifier is set to one, and thus has no effect.


Vector Rotation

This control allows you to rotate the image. Rotation is performed around the Vector Anchor Point (see below), which by default is set to the center of the frame.


Vector Center

This control allows you to position the image within the frame. By clicking on the cross button, you can drag the image in the canvas. The vector center is defined to be the center of the frame before any rotation or movement is performed. If you change the Vector Anchor Point and then rotate the image, the vector center may not line up with the center of the rotated image. This can make it tricky to position.

If you're having difficulty positioning in the Canvas, try zooming out the Canvas (to say, 25%) while you position. That gives you a bigger canvas to position on.


Vector Anchor Point

This control allows you to set the rotation center for the image. By clicking on the cross button, you can drag the anchor point in the canvas. The vector anchor point is defined to be a position in the frame before any scale, rotation, or movement is performed. For this reason, if you intend to modify the position of the vector anchor point, you should do so first, before you make any other transformations. It can be very difficult to modify the vector anchor point position if you have scaled, rotated, or repositioned the image.


Force Update

This control allows you to flush the Final Cut Pro image cache. You should click this control any time you make changes in the Ampede PDF window; otherwise, Final Cut Pro will not register the changes. Force Update has no effect on the rendered image; it does not matter if it is checked or not.
