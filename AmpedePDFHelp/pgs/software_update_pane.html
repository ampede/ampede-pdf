The Software Update Pane

The Software Update pane is where you can check for updated versions of PDF. It requires an Internet connection. No personal information is sent during the process.

You can setup PDF to check for new versions automatically. It will check once a day, and will only check if an Internet connection is already active. It will not connect to the Internet in order to check.

You must launch Final Cut Pro and open a sequence containing a clip with PDF applied in order for PDF to check for a new version.

  • Click on the Check for updates at start-up check box to enable/disable automatic checking for new versions.
