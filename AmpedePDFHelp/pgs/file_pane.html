The File Pane

The File pane is where you choose the file you want PDF to rasterize. If you have not chosen a file, PDF will display a default image.

  • Click on the Choose... button and select the file you want PDF to rasterize in the open dialog that appears.

By default, PDF monitors a file for changes. PDF will give you the opportunity to update itself whenever the file is modified. You can change this behavior.

  • Click on the Monitor file for changes check box to enable/disable file monitoring.

If for some reason PDF does not notice that a file has changed, or you want to update PDF manually, you can.

  • Click on the Reconnect... button and select the version of the file you want PDF to rasterize.

You can also view a file in your default PDF viewer.

  • Click on the Open in viewer... button and the file will open in your default PDF viewer.
