The ColorSync Pane

The ColorSync pane is where you enable PDF's color management feature. By choosing a profile, you can control the colors PDF uses to rasterize the image seen in Final Cut Pro. The profile determines the gamut PDF uses for Final Cut Pro. For options on how PDF maps colors from your source file to Final Cut Pro, view the Quartz Pane help.

  • Click on the Enable color management check box to enable/disable color management.

The Open ColorSync System Preferences button opens the ColorSync Utility which is where you choose the ColorSync profile that PDF will use when color management is enabled. With color management enabled, PDF uses the RGB Default profile. If color management is not enabled, PDF uses your display profile.

You cannot use PDF to adjust the source profile (the profile that describes your pdf file).

Which profile should I choose?

For editors working with video, sRGB is the ColorSync profile to use. It's practically identical to the profiles for DV, SD, and HD video.
