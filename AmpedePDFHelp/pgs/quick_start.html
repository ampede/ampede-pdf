Quick Start Tutorial

Using Ampede's PDF plugin is simple. The first step is to apply PDF to a clip. PDF is located in the Effects > Video Filters > Ampede menu. Once applied, PDF completely overwrites whatever image was on the clip to begin with, so it's best to apply it to a fast clip like Slug.

After applying PDF, you'll see a default image in the viewer. Now is the time to select the Illustrator or PDF file you want PDF to rasterize.

  1. Click on PDF's Options... button visible in the Filters tab of the Viewer window. If you don't see Ampede PDF in the list of filters, try double-clicking the clip you applied PDF to.
  2. The Ampede PDF window appears and the File pane is shown.
  3. Click on the Choose... button and select the file you want PDF to rasterize in the open dialog that appears.
  4. [Optional] Select a page in the file you chose by clicking on the PDF pane's icon and choosing a new page number.
  5. Close the Ampede PDF window.

After you have chosen a file, you'll want to force Final Cut Pro to update its internal render cache. This will free any cached frames it created from the default image.

  • Click on PDF's Force Update checkbox visible in the Filters tab of the Viewer window.

You can now use PDF's controls in the Filters tab of the Viewer window to scale, rotate, and move your file. The controls are fully keyframeable. For a complete description of each control, view the Filter Controls help.

You can also add additional filters to the clip, such as a Guassian blur. Just be sure to add the filters after you apply PDF; otherwise, they won't be used.
