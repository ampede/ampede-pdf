The PDF Pane

In the PDF pane, you set options that determine how PDF understands the file you have chosen and rasterizes it to the screen.


Page Number

PDF can render any page of a PDF file. The selected page is shown rasterized in the bottom of the PDF pane.

To choose a page do one of the following:

  • type the page number in the Page Number field
  • use the up and down arrows next to the Page Number field to select a page


Preserve Aspect Ratio

PDF scales a page to fit fully within the bounds of the frame. By default, it does not change the shape of the page. You can also instruct PDF to stretch the page so that it fills the frame completely.

  • Click the Preserve aspect ratio checkbox to specify whether PDF should preserve the page's shape.


Fill Background

By default, PDF rasterizes your page over an opaque white background. You can tell PDF not to draw the background.

  • Click the Fill background checkbox to enable/disable PDF's background drawing.


Background Color

PDF can draw the background with any color you choose, using Mac OS X's standard color picker.

  1. Click the Background color color box to open the color picker.
  2. Choose a color in the color picker. The background color changes to reflect your selection.
  3. Close the color picker when you are finished


Box Type

A PDF file defines a number of boxes on a page. These boxes are embedded in the PDF or Illustrator file and cannot be directly modified. You can tell PDF what box you want it to render the background color in.

PDF will also let you render the video frame's entire background with the background color. Be aware that the “entire background” is defined before any transformation is calculated; the background will move with the page as you move, rotate, or scale the page.

  • Click the Box Type pop-up menu and select the box you want PDF to fill with the background color.

If you want the background to always be a particular color, regardless of how the PDF file is transformed, you should turn off the background fill and create a new video layer in Final Cut Pro and apply a color to that layer instead. Then, place that layer directly below the layer you have PDF applied to and the background will remain that color.
